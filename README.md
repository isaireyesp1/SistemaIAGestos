# 🧠 Acerca del Sistema

**AI Gesture Detection System** es un sistema de visión por computadora que interpreta gestos humanos capturados por cámara web y los convierte en acciones dentro de una aplicación.

Este proyecto permite construir interfaces sin contacto físico, usando solo movimientos de manos o cuerpo.

El sistema permite:

- 🖐️ Detección de manos en tiempo real
- 🤖 Clasificación de gestos con IA
- ⚡ Ejecución de acciones según el gesto detectado
- 🎮 Integración con juegos o interfaces
- 📊 Visualización del tracking en pantalla
- 🔁 Procesamiento continuo en tiempo real

El proyecto fue desarrollado para practicar:

- Python avanzado
- Computer Vision
- Deep Learning
- Modelos de clasificación
- Procesamiento en tiempo real
- Integración de IA en sistemas interactivos

# 📂 Estructura del Proyecto

```bash
AI-Gesture-System/
│
├── data/               # Dataset de gestos
├── models/             # Modelos entrenados
├── src/
│   ├── main.py         # Punto de entrada
│   ├── camera.py       # Captura de video
```

## 🚀 Instalación

### 1️⃣ Clonar repositorio
```
git clone https://github.com/your-username/ai-gesture-system.git
cd ai-gesture-system
```
### 2️⃣ Instalar dependencias
```
pip install -r requirements.txt
```
### 3️⃣ Ejecutar sistema
```
python src/main.py
```
